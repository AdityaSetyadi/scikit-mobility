Data Structures¶

scikit-mobility provides two different data structures, both extending the Pandas dataframe. TrajDataframe is designed to deal with trajectories, i.e., data in form of GPS points, while FlowDataFrame allows to manage fluxes between places.

mapping(tessellation, remove_na=False)¶

Method to assign to each point of the TrajDataFrame a corresponding tile_id of a given tessellation. :param tessellation: GeoDataFrame containing a tessellation (geometry of points or polygons). :param remove_na: (default False) it removes points that do not have a corresponding tile_id :return: TrajDataFrame with an additional column containing the tile_ids.

plot_diary(user, start_datetime=None, end_datetime=None, ax=None)¶

Requires a TrajDataFrame with clusters, output of preprocessing.clustering.cluster. The column constants.CLUSTER must be present.

Parameters

  • user – str or int user ID whose diary should be plotted.

  • start_datetime – datetime.datetime Only stops made after this date will be plotted. If None the datetime of the oldest stop will be selected.

  • end_datetime – datetime.datetime Only stops made before this date will be plotted. If None the datetime of the newest stop will be selected.

  • ax – matplotlib.axes axes where the diary will be plotted.

Returns

matplotlib.axes of the plotted diary.

plot_stops(map_f=None, max_users=10, tiles='cartodbpositron', zoom=12, hex_color=-1, opacity=0.3, radius=12, popup=True)¶

Requires a TrajDataFrame with stops or clusters, output of preprocessing.detection.stops or preprocessing.clustering.cluster. The column constants.LEAVING_DATETIME must be present.

Parameters

  • map_f – folium.Map folium.Map object where the stops will be plotted. If None, a new map will be created.

  • max_users – int maximum number of users whose stops should be plotted.

  • tiles – str folium’s tiles parameter.

  • zoom – int initial zoom.

  • hex_color – str or int hex color of the stop markers. If -1 a random color will be generated for each user.

  • opacity – float opacity (alpha level) of the stop makers.

  • radius – float size of the markers.

  • popup – bool if True, when clicking on a marker a popup window displaying information on the stop will appear.

Returns

folium.Map object with the plotted stops.

plot_trajectory(map_f=None, max_users=10, max_points=1000, style_function=<function <lambda>>, tiles='cartodbpositron', zoom=12, hex_color=-1, weight=2, opacity=0.75, start_end_markers=True)¶
Parameters

  • map_f – folium.Map folium.Map object where the trajectory will be plotted. If None, a new map will be created.

  • max_users – int maximum number of users whose trajectories should be plotted.

  • max_points – int maximum number of points per user to plot. If necessary, a user’s trajectory will be down-sampled to have at most max_points points.

  • style_function – lambda function function specifying the style (weight, color, opacity) of the GeoJson object.

  • tiles – str folium’s tiles parameter.

  • zoom – int initial zoom.

  • hex_color – str or int hex color of the trajectory line. If -1 a random color will be generated for each trajectory.

  • weight – float thickness of the trajectory line.

  • opacity – float opacity (alpha level) of the trajectory line.

  • start_end_markers – bool add markers on the start and end points of the trajectory.

Returns

folium.Map object with the plotted trajectories.

get_flow(origin_id, destination_id)¶

Get the flow between two locations. If there is no flow between two locations it returns 0. :param origin_id: the id of the origin tile :param destination_id: the id of the tessellation tile :return: The flow between the two locations :rtype: int

plot_flows(map_f=None, min_flow=0, tiles='Stamen Toner', zoom=6, flow_color='red', opacity=0.5, flow_weight=5, flow_exp=0.5, style_function=<function <lambda>>, flow_popup=False, num_od_popup=5, tile_popup=True, radius_origin_point=5, color_origin_point='#3186cc')¶
Parameters

  • map_f – folium.Map folium.Map object where the flows will be plotted. If None, a new map will be created.

  • min_flow – float only flows larger than min_flow will be plotted.

  • tiles – str folium’s tiles parameter.

  • zoom – int initial zoom.

  • flow_color – str color of the flow edges

  • opacity – float opacity (alpha level) of the flow edges.

  • flow_weight – float weight factor used in the function to compute the thickness of the flow edges.

  • flow_exp – float weight exponent used in the function to compute the thickness of the flow edges.

  • style_function – lambda function GeoJson style function.

  • flow_popup – bool if True, when clicking on a flow edge a popup window displaying information on the flow will appear.

  • num_od_popup – int number of origin-destination pairs to show in the popup window of each origin location.

  • tile_popup – bool if True, when clicking on a location marker a popup window displaying information on the flows departing from that location will appear.

  • radius_origin_point – float size of the location markers.

  • color_origin_point – str color of the location markers.

Returns

folium.Map object with the plotted flows.

plot_tessellation(map_osm=None, maxitems=-1, style_func_args={}, popup_features=['tile_ID'], tiles='Stamen Toner', zoom=6, geom_col='geometry')¶
Parameters

  • map_osm – folium.Map folium.Map object where the GeoDataFrame gdf will be plotted. If None, a new map will be created.

  • maxitems – int maximum number of tiles to plot. If -1, all tiles will be plotted.

  • style_func_args – dict dictionary to pass the following style parameters (keys) to the GeoJson style function of the polygons: ‘weight’, ‘color’, ‘opacity’, ‘fillColor’, ‘fillOpacity’

  • popup_features – list when clicking on a tile polygon, a popup window displaying the information in the columns of gdf listed in popup_features will appear.

  • tiles – str folium’s tiles parameter.

  • zoom – int initial zoom.

  • geom_col – str name of the geometry column of gdf.

Returns

folium.Map object with the plotted GeoDataFrame.
